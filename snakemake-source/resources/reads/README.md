# Input reads (placeholder)

This directory is where the raw FASTQ files referenced by `config/samples.tsv`
live. They are **not** committed to the repository.

For a smoke test you can drop in a handful of tiny gzipped FASTQ files named to
match the sample sheet, e.g.:

```
CEH_pond_01_R1.fastq.gz
CEH_pond_01_R2.fastq.gz
...
```
